## instalation & Setup
Clone the repository,
```bash
git clone https://github.com/pashampavan/i-NoteBook.git
```
Next, install the dependencies:
```bash
npm install
```
Then, run the front-end development server:
```bash
npm run start
```
## Usage
1. Open website.
2. Enter text to analyse and do some more operations.
